# cc-programs
Random programs made myself or by others for ComputerCraft
----------------------------------------------------------
Current programs list :

- Advanced Tunnel (AdvancedTunnelv1.3.1.lua, AdvancedTunnelv1.3.4.lua)
made by Henness.
enhanced version of the basic tunnel program available in the turtle, allows you to choose the tunnel size, automatically places torches, and displays the mining results.

- Clock (Clock.lua)
displays the current time on a screen.

- Crop Farm (CropFarm.lua)
made by Gaarnik.
allows a turtle to sow and harvest crops.

- Monitor (Monitor.txt)
test programs for the monitor. useless.

- Simple Tunnel (SimpleTunnel.lua)
tunnel program.

- Solar Factory (SolarFactory.lua)
program to craft IC2 solar pannels.

- Solar Factory 2 (SolarFactory2.lua)
another program to craft IC2 solar pannels.

- Tree Farm (TreeFarmv0.8.lua, TreeFarmv1.0.lua)
allows a turtle to harvest trees.

- Tree Farm 2 (TreeFarm2.lua)
another tree farm.
